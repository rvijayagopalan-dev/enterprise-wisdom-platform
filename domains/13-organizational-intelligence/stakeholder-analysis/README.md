# Sub-Domain: Stakeholder Analysis

## Purpose
Identify all stakeholders affected by a transformation, assess their influence and interest, and design targeted engagement strategies.

## Analysis Dimensions
- Stakeholder identification and mapping
- Power / Interest grid positioning
- Current vs. desired stance assessment
- Engagement strategy per stakeholder
- Communication plan design

## Outputs
- Stakeholder Map
- Power/Interest Grid
- Stakeholder Engagement Plan
- Communication Matrix
