# Sub-Domain: Organizational Network Analysis

## Purpose
Map informal influence networks and information flows to identify hidden influencers, bottlenecks, and collaboration gaps.

## Analysis Types
- Information flow analysis
- Influence network mapping
- Collaboration pattern analysis
- Silo detection
- Key connector and broker identification

## Outputs
- Informal Network Map
- Influence Topology Diagram
- Collaboration Gap Analysis
- Change Champion Identification
