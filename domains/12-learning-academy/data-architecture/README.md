# Learning Track: Data Architecture

## Purpose
Develop expertise in data strategy, data mesh, data products, and modern data platform architecture.

## Modules
- Data strategy and roadmap development
- Data Mesh principles and implementation
- Data product design and management
- Data governance and quality
- Modern data platform architecture (Databricks, Snowflake, BigQuery)
- Knowledge graph and GraphRAG design

## Certifications
- Data Architecture Practitioner
- Data Mesh Specialist
