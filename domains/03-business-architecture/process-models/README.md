# Sub-Domain: Process Models

## Purpose
Document, analyze, and optimize business processes using standard modeling notations.

## Frameworks
- BPMN 2.0 — process flow modeling
- APQC PCF — process classification and benchmarking
- Lean / Six Sigma — waste elimination

## Capabilities
- Current-state process documentation
- Future-state process design
- Process performance measurement
- AI-assisted process optimization

## Outputs
- BPMN Process Diagrams
- Process Performance Baselines
- Automation Opportunity Maps
