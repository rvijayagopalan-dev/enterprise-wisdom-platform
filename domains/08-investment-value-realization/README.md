# Domain 08: Investment & Value Realization

## Purpose
Build business cases, optimize investment portfolios, and track value realization from transformation programs.

## Sub-Domains

| Sub-Domain | Description |
|------------|-------------|
| [Business Case](./business-case/README.md) | AI-generated business cases with financial models |
| [Investment Planning](./investment-planning/README.md) | Investment prioritization and portfolio planning |
| [Portfolio Optimization](./portfolio-optimization/README.md) | Optimize the mix of initiatives for maximum return |
| [Benefits Realization](./benefits-realization/README.md) | Track and measure realized benefits against plan |

## Financial Metrics
- **ROI** — Return on Investment
- **NPV** — Net Present Value
- **IRR** — Internal Rate of Return
- **Payback Period** — Time to recover investment
- **TCO** — Total Cost of Ownership

## Outputs
- Executive Financial Models
- Investment Prioritization Matrix
- Benefits Realization Reports
- Executive Financial Dashboards

## AI Advisor
**CFO Advisor** — evaluates investment proposals, models financial scenarios, and recommends portfolio allocation.
