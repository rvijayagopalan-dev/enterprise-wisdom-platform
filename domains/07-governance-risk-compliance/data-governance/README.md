# Sub-Domain: Data Governance (GRC)

## Purpose
Govern data assets to ensure quality, lineage, ownership, and regulatory compliance across the enterprise.

## Controls
- Data ownership and stewardship policies
- Data classification and sensitivity labeling
- Privacy and consent management
- Regulatory data compliance (GDPR, CCPA, HIPAA)

## Outputs
- Data Governance Policy
- Data Classification Framework
- Privacy Impact Assessments
