# Sub-Domain: Agentic AI

## Purpose
Design and govern autonomous AI agents that can reason, plan, and execute multi-step tasks with minimal human intervention.

## Frameworks
- LangGraph — stateful agent orchestration
- CrewAI — role-based multi-agent systems
- Semantic Kernel — enterprise agent integration
- MCP — Model Context Protocol for agent tool use

## Design Areas
- Agent architecture patterns
- Tool and memory design for agents
- Human-in-the-loop control points
- Agent observability and monitoring

## Outputs
- Agentic AI Architecture Blueprint
- Agent Design Patterns
- Agent Governance Policy
