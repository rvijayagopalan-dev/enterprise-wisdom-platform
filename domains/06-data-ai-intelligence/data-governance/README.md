# Sub-Domain: Data Governance

## Purpose
Establish policies, standards, and controls to ensure data quality, lineage, ownership, and regulatory compliance.

## Capabilities
- Data ownership and stewardship assignment
- Data quality rules and monitoring
- Data lineage tracking
- Metadata management and cataloging
- Regulatory compliance (GDPR, CCPA, HIPAA)

## Outputs
- Data Governance Framework
- Data Quality Scorecards
- Data Lineage Maps
- Data Catalog
