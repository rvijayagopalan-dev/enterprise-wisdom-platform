# Sub-Domain: Knowledge Graphs

## Purpose
Build enterprise knowledge representation systems that connect entities, relationships, and context to enable AI reasoning.

## Technology
- Neo4j — graph database
- GraphRAG — graph-augmented retrieval
- RDF / OWL — semantic knowledge modeling
- Vector Search — semantic similarity retrieval

## Use Cases
- Enterprise Memory Graph (decisions, risks, artifacts)
- Capability-to-strategy traceability graph
- Technology dependency graph
- Industry and competitive intelligence graph

## Outputs
- Enterprise Knowledge Graph Schema
- Graph Population Pipeline
- GraphRAG Query Layer
