# Sub-Domain: Workforce Planning

## Purpose
Plan workforce changes required to support transformation — covering hiring, reskilling, redeployment, and attrition management.

## Planning Dimensions
- Current workforce inventory (roles, skills, headcount)
- Future workforce requirements from TOM and strategy
- Gap analysis (roles to hire, skills to develop, roles to retire)
- Transition planning and timeline

## Outputs
- Workforce Transformation Plan
- Hiring Plan
- Reskilling Investment Plan
- Workforce Transition Timeline
