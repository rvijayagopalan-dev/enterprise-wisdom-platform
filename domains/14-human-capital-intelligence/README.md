# Domain 14: Human Capital Intelligence

## Purpose
Connect business capabilities to the people, roles, and skills required to deliver them — and identify gaps that must be addressed through hiring, training, or reskilling.

## The Missing Layer
Almost no enterprise architecture or strategy platform understands skills. This domain bridges capabilities to workforce, making it possible to answer:
> "What skills are missing for our AI transformation?"

## Sub-Domains

| Sub-Domain | Description |
|------------|-------------|
| [Skills Graph](./skills-graph/README.md) | Map capabilities to roles and skills required |
| [Learning Engine](./learning-engine/README.md) | Recommend training, reskilling, and development paths |
| [Workforce Planning](./workforce-planning/README.md) | Plan workforce changes to support transformation |

## Skills Connection Model
```
Capabilities
      ↓
Processes
      ↓
Roles
      ↓
Skills
```

## Outputs
- Skills Gap Analysis
- Workforce Transformation Plan
- Learning & Development Recommendations
- Hiring Strategy
