# Sub-Domain: Regulatory Forces

## Purpose
Monitor regulatory developments that affect enterprise strategy, data practices, and AI deployment.

## Categories

| Category | Regulations |
|----------|------------|
| Privacy | GDPR, CCPA, State Privacy Laws |
| AI | EU AI Act, NIST AI RMF, ISO 42001 |
| Financial | SOX, Basel, Dodd-Frank |
| Industry | PCI DSS, HIPAA |

## Outputs
- Regulatory Change Impact Assessment
- Compliance Gap Analysis
- Strategic Constraint Flags
