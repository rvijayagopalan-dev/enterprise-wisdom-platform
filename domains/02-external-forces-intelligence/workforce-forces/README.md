# Sub-Domain: Workforce Forces

## Purpose
Monitor labor market trends that affect hiring strategy, operating models, and automation priorities.

## Trends
- Skills availability (especially AI/ML skills)
- AI skills shortage
- Remote and hybrid work adoption
- Workforce demographics

## Impact Areas
- Hiring strategy
- Operating model design
- Automation and AI deployment strategy

## Outputs
- Workforce Trend Report
- Skills Gap Forecast
- Automation Opportunity Analysis
