# Domain 02: External Forces Intelligence

## Purpose
Continuously monitor factors outside the enterprise that affect strategy — the starting point for all strategic reasoning.

## Model
```
External Forces
      +
Internal Drivers
      ↓
Strategy Generation → Strategic Themes → Initiatives → Capabilities → Architecture → Investments → Outcomes
```

## Sub-Domains

| Sub-Domain | Description |
|------------|-------------|
| [Macroeconomic Forces](./macroeconomic-forces/README.md) | GDP, inflation, interest rates, employment |
| [Industry Forces](./industry-forces/README.md) | Competitive dynamics via Porter's Five Forces |
| [Technology Forces](./technology-forces/README.md) | AI, cloud, data, security technology radar |
| [Regulatory Forces](./regulatory-forces/README.md) | Privacy, AI regulation, financial, industry compliance |
| [Geopolitical Forces](./geopolitical-forces/README.md) | Trade policies, tariffs, sanctions, regional risks |
| [Market Forces](./market-forces/README.md) | Customer behavior, demographics, digital adoption |
| [Workforce Forces](./workforce-forces/README.md) | Skills availability, AI talent, remote work trends |
| [Ecosystem Forces](./ecosystem-forces/README.md) | Vendors, partners, SaaS providers, open source |
| [Sustainability & ESG](./sustainability-esg/README.md) | Carbon, energy, ESG reporting, sustainable supply chain |
| [Disruption Intelligence](./disruption-intelligence/README.md) | New business models, AI-native competitors, industry convergence |

## Scoring Engine
Every force is scored across 5 dimensions:

| Dimension | Score |
|-----------|-------|
| Opportunity | 1–10 |
| Threat | 1–10 |
| Urgency | 1–10 |
| Strategic Impact | 1–10 |
| Financial Impact | 1–10 |

## Outputs
- Opportunity Analysis
- Threat Analysis
- Strategic Impact Assessment
- External Forces Dashboard
