# Sub-Domain: Capability Twin

## Purpose
Maintain a live digital model of all business capabilities, their maturity, performance, and investment levels.

## Model Attributes
- Capability name and hierarchy
- Current maturity level (1–5)
- Target maturity level
- Investment allocated
- Strategic importance score
- Gap score (target - current)

## Simulation Questions
- What capabilities are impacted by this initiative?
- Which capability gaps are blocking strategic objectives?

## Outputs
- Live Capability Heat Map
- Capability Gap Analysis
- Investment-to-Capability Impact Model
